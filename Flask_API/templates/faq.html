FAQ

Welcome to the FAQ page for our AI-powered crop harvesting prediction test application. Our application is designed to help test the readiness of trained models. If you have any questions about how our test application works or how it can benefit your farming operation, please read on for more information.

What types of crops can the AI test application predict harvest readiness for?

The AI has been trained with 4 different lettuce species: Satine, Salanova, Aphylion and Lugano.

How does the AI test application determine the optimal harvest time for crops?

By extracting features like the height of the plant from uploaded images the AI is able to use these features as variables in deciding the optimal harvest date.

How accurate are the harvest predictions generated by the AI test application?

The application uses multiple models. So there is no defined accuracy.
